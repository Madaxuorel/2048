Pour lancer le jeu, executez 2048.exe, situé dans la dossier "jeu".

Vous n'avez pas à installer d'extension, les fichiers nécessaires sont déjà inclus dans le jeu.

Le Jeu est fait pour fonctionner UNIQUEMENT avec Windows

Le code source pour le menu, le mode 1 joueur et le mode 2 joueur est situé dans le fichier "Code source".




COMMANDES :

-Déplacement avec fléche haute , bas , droite et gauche
- touche 'r' pour reset la partie
- touche 's'pour sauvegarder
- touche échap pour quitte
